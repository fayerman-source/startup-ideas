# Startup Ideas Worth Validating

A curated, monthly-updated list of **real problem spaces worth validating**, surfaced from public pain signals across the web.

Each entry is a raw *seed*: a genuine problem someone has, tagged with the operator persona who feels it. A seed is the starting point, not the answer. **[1mil.app](https://1mil.app)** turns any seed into a research-backed, ranked, operator-fit business opportunity matched to your background.

## How to use this

1. **Browse** the latest batch in [`seeds/`](./seeds/).
2. **Click a seed** that fits a market or skill set you know.
3. **Validate it** in [1mil.app](https://1mil.app) (the link is pre-filled with the problem and persona) to find out whether it's worth building.
